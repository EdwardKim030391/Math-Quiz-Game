How to Play

  • Start the Game: Select a difficulty level: Easy, Medium, Hard, or Extreme.
  • Solve the Problems: All questions are math related. Solve the math problems that appear on the screen.
  • Time Limit: There is a time limit for each mode.
  • Check Your Score: You earn points for each correct answer, and you need to answer all 10 questions correctly.
  • Game Over: If you answer all the questions within the time limit, you win. If not, a "Game Over" message will appear.
